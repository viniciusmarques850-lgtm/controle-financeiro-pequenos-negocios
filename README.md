# 💰 Controle Financeiro para Pequenos Negócios

> Sistema simples de controle de entradas, saídas e relatórios automáticos, 
> pensado para pequenos negócios que ainda controlam finanças em caderno ou planilha solta.

## 🎯 O problema

Muitos pequenos negócios não têm visão clara de quanto realmente sobra 
no fim do mês. O controle manual (caderno, planilha solta) dificulta 
identificar onde está o lucro e onde está o prejuízo.

## ✅ O que o sistema faz (ou vai fazer)

- [ ] Cadastro de entradas e saídas
- [ ] Categorização de lançamentos
- [ ] Relatório mensal automático (gráfico)
- [ ] Resumo de lucro/prejuízo por categoria
- [ ] Exportação em PDF

## 🛠️ Tecnologias

- Python
- SQLite
- Pandas
- Matplotlib / Streamlit (interface)
